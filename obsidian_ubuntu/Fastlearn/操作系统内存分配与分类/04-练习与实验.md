# 练习与实验：操作系统内存分配与分类

## 主动回忆

- 闭卷写出进程地址空间各区域及其典型内容。
- 闭卷讲清 `malloc -> 首次访问 -> page fault -> 建立映射` 链路。
- 闭卷区分匿名页、文件页、页缓存、swap。

## 实现练习

- 用伪代码写一个最简伙伴系统的“拆分 + 合并”逻辑。
- 用 C/C++ 写一个小程序，对比小块分配与大块 `mmap` 分配时的 RSS 变化。

## 调试练习

- 给出 `top`、`vmstat`、`pmap`、`smaps` 片段，判断更像泄漏、碎片还是页缓存膨胀。
- 解释 `minor fault` 突增和 `major fault` 突增分别说明什么。

## 设计练习

- 设计一个高 QPS 服务的内存观测面板，说明你要看哪些指标，以及它们如何关联到故障定位。
- 讨论缓存系统为什么既依赖内存局部性，又容易被 NUMA 和大页策略影响。

## 对比练习

- 比较 `brk` 与 `mmap`。
- 比较伙伴系统与 slab/slub。
- 比较内部碎片与外部碎片。
