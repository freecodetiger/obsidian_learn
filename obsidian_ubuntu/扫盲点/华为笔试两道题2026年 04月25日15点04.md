### 题目名称：软件包依赖解析与循环检测

#### 📝 题目描述

在复杂的软件系统中，经常会遇到软件包之间的依赖关系。为了保证系统构建的稳定性，包管理工具需要处理两个核心问题：

1. **版本统一（向下兼容）**：如果系统中有多个包依赖了同一个包的不同版本，为了兼容性，系统必须统一采用该被依赖包的**最高版本**（最新版本）。
    
2. **循环依赖检测**：包与包之间的依赖链路不能形成闭环（例如：A 依赖 B，B 依赖 C，C 又依赖 A）。如果存在循环依赖，系统将发生死锁，无法成功构建。

给定若干条包的依赖规则，请你编写程序检查这些依赖是否存在循环引用。如果不存在循环引用，请将所有的依赖规则统一升级到各自被依赖包的最高版本，并按原顺序输出；如果存在循环引用，则直接报错。

_(注：本题固定包含两组测试数据连续输入)_
#### 📥 输入格式

输入包含固定两组测试数据。每组测试数据的格式如下：

- 第一行是一个整数 `k`，表示接下来有 `k` 条依赖关系。
    
- 接下来的 `k` 行，每行包含三个整数，以英文逗号 `,` 分隔，格式为 `a,b,c`：
    
    - `a` 表示依赖方的包 ID（即当前包）。
        
    - `b` 表示被依赖方的包 ID（即需要引入的包）。
        
    - `c` 表示包 `a` 依赖包 `b` 时指定的版本号。
#### 📤 输出格式

对于每组测试数据：

- 如果检测到系统存在循环依赖，请直接输出字符串 `false` 并换行。
    
- 如果不存在循环依赖，请**按照输入的原始顺序**，逐行输出更新版本号后的依赖关系，格式同样为 `a,b,max_c`，其中 `max_c` 是所有依赖规则中对包 `b` 要求的最高版本号。

            if (indeg[x] == 0) q.push(x);

        int popped = 0;
        while (!q.empty()) {
            int u = q.front();
            q.pop();
            ++popped;

            auto it = a_b.find(u);
            if (it == a_b.end()) continue;

            for (int v : it->second) {
                if (--indeg[v] == 0) q.push(v);
            }
        }

        // 有环：无法弹出所有出现过的节点
        if (popped != (int)nodes.size()) {
            cout << "false" << '\n';
            continue;
        }

        // 无环：输出 a,b,n_c（c 为 b 的最新版本）
        // 按输入顺序输出；如果不要求顺序，也可以遍历 a_b
        for (const auto &e : edges) {
            int a = e.first, b = e.second;
            cout << a << "," << b << "," << maxVer[b] << '\n';
        }
    }

    return 0;

### 🗺️ 题目名称：迷宫最少拐弯寻路

#### 📝 题目描述

给定一个 $a \times b$ 的二维网格（迷宫），其中 `0` 表示可以通行的空地，非 `0` 的数字（通常为 `1`）表示不可通行的障碍物。

你需要从网格的左上角起点 `(0, 0)` 走到右下角终点 `(a-1, b-1)`。

在移动过程中，**你只能向右或向下移动**。

每次你改变移动方向（从“向右”变为“向下”，或者从“向下”变为“向右”）时，记作一次“拐弯”。

请计算并返回从起点到达终点所需的**最少拐弯次数**。

如果无法到达终点，或者起点/终点本身就是障碍物，或者给定的地图尺寸不合法，请输出 `-1`。

#### 📥 输入格式

- 第一行包含两个整数 $a$ 和 $b$，分别代表网格的行数和列数。
    
- 接下来的 $a$ 行，每行包含 $b$ 个整数（以空格分隔），代表网格的布局。
    

#### 📤 输出格式

- 输出一个整数，表示到达目标位置的最少拐弯次数。
    
- 如果无解或输入不合法，输出 `-1`。
    

#### ⚠️ 数据范围与约束

- $1 \le a, b \le 100$
    

---

#### 💡 输入输出样例

**输入：**

Plaintext

0 0 1

0 0 0
```

**输出：**

**样例解释：**

最少拐弯路径为：

1. `(0,0)` -> `(0,1)` (向右，不拐弯)
    
2. `(0,1)` -> `(1,1)` (向下，**拐弯 1 次**)
    
3. `(1,1)` -> `(1,2)` (向右，**拐弯 1 次**)
    
4. `(1,2)` -> `(2,2)` (向下，**拐弯 1 次**)
    
    总共需要 3 次拐弯。

### 🧠 代码核心解法拆解（状态机 DP）

代码中最精妙的地方在于它不是简单地用 `dp[i][j]` 记录最小步数，而是**将到达每个格子的“最后一步的方向”作为了状态的一部分**：

- `dpR[i][j]`：到达位置 `(i,j)` 且最后一步是**向右**走时的最小拐弯数。
    
- `dpD[i][j]`：到达位置 `(i,j)` 且最后一步是**向下**走时的最小拐弯数。
    

状态转移非常符合直觉：

- 如果想**向右**走到 `(i,j)`，可以是从 `(i, j-1)` 继续向右（拐弯数不变：`dpR[i][j-1]`），也可以是从 `(i, j-1)` 先向下走再向右（发生拐弯，拐弯数+1：`dpD[i][j-1] + 1`）。
    
- 同理，**向下**走到 `(i,j)`，可以是顺着原方向继续向下（不加转弯数），也可以是从向右的状态转过来（转弯数+1）。
只允许向右向下走，拐弯次数最少多少次到右下角，如果不能到达返回 false
```
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int a, b;
    cin >> a >> b;
    if (a <= 0 || b <= 0 || a > 100 || b > 100) {
        cout << -1 << "\n";
        return 0;
    }

    vector<vector<int>> arr(a, vector<int>(b));
    for (int i = 0; i < a; ++i) {
        for (int j = 0; j < b; ++j) cin >> arr[i][j];
    }

    if (arr[0][0] != 0 || arr[a - 1][b - 1] != 0) {
        cout << -1 << "\n";
        return 0;
    }

    // 特判单点
    if (a == 1 && b == 1) {
        cout << 0 << "\n";
        return 0;
    }

    const int INF = 1e9;
    // dpR[i][j]: 到(i,j)且最后一步是“向右”时的最小转弯数
    // dpD[i][j]: 到(i,j)且最后一步是“向下”时的最小转弯数
    vector<vector<int>> dpR(a, vector<int>(b, INF));
    vector<vector<int>> dpD(a, vector<int>(b, INF));

    // 初始化从起点走一步
    if (b > 1 && arr[0][1] == 0) dpR[0][1] = 0;
    if (a > 1 && arr[1][0] == 0) dpD[1][0] = 0;

    for (int i = 0; i < a; ++i) {
        for (int j = 0; j < b; ++j) {
            if (arr[i][j] != 0) continue;
            if (i == 0 && j == 0) continue;

            // 从左侧到当前：最后一步是向右
            if (j - 1 >= 0 && arr[i][j - 1] == 0) {
                dpR[i][j] = min(dpR[i][j], dpR[i][j - 1]);       // 继续向右，不转弯
                dpR[i][j] = min(dpR[i][j], dpD[i][j - 1] + 1);   // 下->右，转弯+1
            }

            // 从上侧到当前：最后一步是向下
            if (i - 1 >= 0 && arr[i - 1][j] == 0) {
                dpD[i][j] = min(dpD[i][j], dpD[i - 1][j]);       // 继续向下，不转弯
                dpD[i][j] = min(dpD[i][j], dpR[i - 1][j] + 1);   // 右->下，转弯+1
            }
        }
    }

    int ans = min(dpR[a - 1][b - 1], dpD[a - 1][b - 1]);
    if (ans >= INF) cout << -1 << "\n";
    else cout << ans << "\n";

    return 0;
}
